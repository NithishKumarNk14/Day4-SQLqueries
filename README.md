# SQL for Data Analysis - Data Analyst Internship Task 4

## Overview:

This task demonstrates my ability to manipulate and analyze structured e-commerce data using SQL. I used a sample relational dataset with customers, products, orders, and order items to write queries that extract meaningful insights using joins, subqueries, aggregate functions, and views.

---

## Tables Used:

1. **customers** – stores customer information (ID, name, email, country)
2. **products** – contains product details (ID, name, category, price)
3. **orders** – stores order-level details (order ID, customer ID, date, total)
4. **order_items** – item-level details of products in each order
